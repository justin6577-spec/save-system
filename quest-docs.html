QuestSystem — Documentation

Data-driven quest and mission tracking. Quest chains, objectives, rewards, and full Blueprint support. GameInstanceSubsystem — drop in and go.

Installation

  1. Copy QuestSystem/ into Plugins/
  2. Add "QuestSystem" to .uproject Plugins array
  3. Regenerate project files + build
Requirement: UE 5.4+. GameInstanceSubsystem — auto-initializes.

Quick Start (BP)

Open any Blueprint, search for the subsystem/component, and call the main API functions. All functions are BlueprintCallable.

Quick Start (C++)

#include "QuestSystem.h"
// Get the subsystem/component and call Start/Initialize functions

Features

  • Data-driven quest definitions
  • 4 objective types (Collect/Kill/Reach/Interact)
  • quest chains with prerequisites
  • repeatable quests
  • auto-complete tracking
  • 3 Blueprint events (OnQuestStateChanged/OnObjectiveProgress/OnQuestCompleted)
  • reward dispatch

API

Full BlueprintCallable API. All public functions exposed to Blueprints.

Technical

  • Engine: 5.4–5.8
  • Platforms: Windows, Linux, Mac
  • Module: QuestSystem (Runtime)
  • Classes: 6 (UQuestManager, FQuestDefinition, FQuestObjective, FQuestReward, FActiveQuest, FQuestSystemModule)
  • Network: No
  • Blueprints: 0 (BlueprintCallable API)
