PoolingSystem — Documentation

High-performance object and actor pooling for Unreal Engine 5.4–5.8. Reduces spawn/destroy overhead for projectiles, enemies, VFX, and any frequently-created objects.

Installation

  1. Copy PoolingSystem/ into Plugins/
  2. Add "PoolingSystem" to .uproject Plugins array
  3. Regenerate project files + build
Requirement: UE 5.4+. No external dependencies.

Quick Start (BP)

  1. Call Get Pool Subsystem
  2. Call Spawn Pooled Actor with your actor class
  3. Call Return To Pool on the actor when done

Quick Start (C++)

#include "PoolSubsystem.h"

UPoolSubsystem* Pool = GetWorld()->GetSubsystem<UPoolSubsystem>();
APooledProjectile* Proj = Pool->SpawnPooledActor<APooledProjectile>(Class);

// ... use projectile ...

Proj->ReturnToPool();

Classes

ClassPurpose
UPoolSubsystemWorld subsystem — main pooling API
UPoolConfigPer-class pool configuration (size, warmup)
IPoolableInterface for pool-compatible actors

API

// Spawn from pool (creates new if pool empty)
T* SpawnPooledActor(UClass* Class, FTransform Transform);

// Return actor to pool
void ReturnToPool(AActor* Actor);

// Pre-warm pool with N instances
void WarmPool(UClass* Class, int32 Count);

// Configure pool per-class
void SetPoolConfig(UClass* Class, FPoolConfig Config);
