SpawnSystem — Documentation

Configurable wave-based actor spawning with timers, formations, and placement strategies. Ideal for enemy waves, loot drops, and procedural population.

Installation

  1. Copy SpawnSystem/ into Plugins/
  2. Add "SpawnSystem" to .uproject Plugins array
  3. Regenerate project files + build
Requirement: UE 5.4+. WorldSubsystem — auto-initializes per world.

Quick Start (BP)

Open any Blueprint, search for the subsystem/component, and call the main API functions. All functions are BlueprintCallable.

Quick Start (C++)

#include "SpawnSystem.h"
// Get the subsystem/component and call Start/Initialize functions

Features

  • Wave-based spawning
  • 4 spawn patterns (Random/Sequential/Circle/Grid)
  • configurable timers and delays
  • max actor limits
  • auto-loop waves
  • single-actor spawn
  • multi-point spawn
  • 4 Blueprint events

API

Full BlueprintCallable API. All public functions exposed to Blueprints.

Technical

  • Engine: 5.4–5.8
  • Platforms: Windows, Linux, Mac
  • Module: SpawnSystem (Runtime)
  • Classes: 5 (USpawnManager, FWaveDefinition, FSpawnConfig, ESpawnPattern, FSpawnSystemModule)
  • Network: No
  • Blueprints: 0 (BlueprintCallable API)
