SimpleInventory — Documentation

Lightweight inventory component for Unreal Engine 5.4–5.8. Add/remove items, stack management, weight tracking, and full Blueprint support.

Installation

  1. Copy SimpleInventory/ into Plugins/
  2. Add "SimpleInventory" to .uproject Plugins array
  3. Regenerate project files + build
Requirement: UE 5.4+. No external dependencies.

Quick Start (C++)

#include "InventoryComponent.h"
UInventoryComponent* Inv = Actor->FindComponentByClass<UInventoryComponent>();
Inv->AddItem(FName("Gold"), 100);
int32 Count = Inv->GetItemCount(FName("Gold"));

Features

  • Add/remove items by ID with stack size limits
  • Item weight tracking with max capacity
  • HasItem/GetItemCount/GetAllItems queries
  • BlueprintCallable API — full Blueprint support
  • Single ActorComponent — add to any actor

API

bool AddItem(FName ItemID, int32 Count);
