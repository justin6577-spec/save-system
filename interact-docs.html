Interaction System — Documentation

Raycast-based object interaction for UE 5.4–5.8. Outline highlights, prompts, hold-to-interact. Works with any actor.

Installation

  1. Copy InteractionSystem/ into Plugins/
  2. Add "InteractionSystem" to .uproject Plugins array
  3. Regenerate project files + build
Requirement: UE 5.4+. UMG for interaction widget.

Quick Start (BP)

  1. Add InteractionComponent to your player character
  2. Add Interactable interface to any actor (doors, items, NPCs)
  3. System auto-detects, highlights, and shows prompts

Quick Start (C++)

#include "InteractionComponent.h"

UInteractionComponent* Interact = Player->FindComponentByClass<UInteractionComponent>();
// System auto-runs — just add IInteractable to target actors

How It Works

  1. Detect: Line trace + overlap from camera every frame
  2. Highlight: Custom depth outline on focused actor
  3. Prompt: Interaction widget shows action text + key hint
  4. Execute: Hold-to-interact (configurable duration) or instant

Classes

ClassPurpose
