Health & Damage System — Documentation

Drop-in health, damage, armor, and regen system for UE 5.4–5.8. Full Blueprint support with 7 events.

Installation

  1. Copy HealthSystem/ into Plugins/
  2. Add "HealthSystem" to .uproject Plugins array
  3. Regenerate project files + build
Requirement: UE 5.4+. UMG module for health bar widget.

Quick Start (BP)

  1. Add HealthComponent to any actor
  2. Call Apply Damage to deal damage
  3. Use events: OnDamaged, OnDeath, OnHealed

Quick Start (C++)

#include "HealthComponent.h"

UHealthComponent* Health = Actor->FindComponentByClass<UHealthComponent>();
Health->ApplyDamage(25.0f, EDamageType::Fire);

Damage Types

  • Physical, Fire, Ice, Poison, Magic
  • Armor can have per-type resistance values

Classes

ClassPurpose
UHealthComponentActor component — health, damage, death
UArmorComponentDamage reduction per type
UHealthRegenComponentAuto-health regeneration

Events (BlueprintAssignable)

  • OnDamaged, OnDeath, OnHealed, OnArmorChanged, OnMaxHealthChanged, OnRevived, OnDamageTypeChanged
